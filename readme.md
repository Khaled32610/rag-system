# 14-Day AI Engineer Plan: RAG System

This project is built as part of a 14-day intensive plan to master AI engineering through building real-world projects. 

## Overview
A Retrieval-Augmented Generation (RAG) system that allows an LLM to answer questions based on custom document inputs.

## Tech Stack
* **Python**
* **LangChain**
* **FastAPI**
* **FAISS** (Coming soon)
* **Docker** (Coming soon)

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`
3. Activate the environment.
4. Install dependencies: `pip install -r requirements.txt`
